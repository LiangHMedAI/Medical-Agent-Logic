### 🚀 今日逻辑实现：药物安全自动审计
**场景演示**：识别患者潜在用药风险

* **输入药物**：布洛芬 (Ibuprofen)
* **患者背景**：胃溃疡病史 (Gastric Ulcer)
* **判定逻辑**：
    * **结论**：🚨 **禁用 (Contraindicated)**
    * **医学原理**：布洛芬会刺激胃酸分泌，极易诱发消化道出血或加重溃疡。
    * **处置指令**：切勿自行服用，必须拦截处方并提示咨询医师。
